## Gym Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/1b848dc989a54b22b6063ad0513d88c0/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User (Member), Staff, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Member Management
6. Membership Card Management
7. Coach Information Management
8. Fitness Project Management
9. Fitness Equipment Management
10. Fitness Activity Management



**Staff Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Membership Card Management
6. Coach Information Management
7. Fitness Project Management
8. Guidance Project Management
9. Fitness Equipment Management
10. Fitness Activity Management



**Administrator Functions:**
1. Login
2. Change Password
3. Personal Information
4. Member Management
5. Staff Management
6. Membership Card Management
7. Membership Card Type Management
8. Coach Information Management
9. Dismissal Management
10. Fitness Project Management
11. Guidance Project Management
12. Fitness Equipment Management
13. Fitness Activity Management
